# 叙事时空地图

  一个面向小说/故事创作者的叙事管理工具，帮助统筹管理复杂的故事要素。

  ## 快速开始

  ``bash
  npm install
  
  npm run dev
  
  浏览器打开 http://localhost:5173 即可使用。

  功能

  - 人物管理 — 姓名、称号、状态、动机、弧光、高清图片（支持按时间段切换外观）、置顶、拖拽出场
  - 地点地图 — 双击添加地点，拖拽布局，支持按时间段切换背景图
  - 事件时间线 — 关联人物/地点/事物/时代/章节，时间轴拖拽浏览
  - 势力系统 — 派系创建，成员与领地关联
  - 章节管理 — 规划→写作→完成 进度追踪
  - 伏笔追踪 — 埋设与回收状态
  - 世界观设定 — 按类别整理的键值对设定
  - 草稿笔记 — 自由文本，随时记录灵感
  - 关系图谱 — 人物/事物/地点/势力之间的关联
  - 导入导出 — JSON 存档、Markdown 大纲导出
